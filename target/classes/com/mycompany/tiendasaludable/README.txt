EP1.1: Análisis de los datos y funcionalidades

El proyecto propuesto es un sistema de gestión de ventas para una tienda. Los datos que se utilizarán son los siguientes:

Clientes: nombre, apellido, dirección, teléfono, correo electrónico y fecha de nacimiento.
Productos: nombre, descripción, precio y cantidad en stock.
Ventas: fecha, cliente, producto y cantidad vendida.
Las principales funcionalidades que se deben implementar son:

Agregar, modificar y eliminar clientes.
Agregar, modificar y eliminar productos.
Registrar ventas y actualizar el stock de productos.
Generar reportes de ventas por periodo, cliente y producto.
Permitir la búsqueda de clientes y productos por nombre y/o descripción.
